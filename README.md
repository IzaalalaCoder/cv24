# Sujet

Il s'agit d'un apprentissage du framework SpringBoot tout en manipulant les CV. On peut donc :
- Visualiser un CV 
- Visualiser tous les CV disponible
- Ajouter un CV
- Supprimer un CV
- Rechercher un CV selon un critère de recherche
